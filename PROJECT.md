# BestyrelsesApp

## Vision

En AI-drevet platform til bestyrelser, udvalg og foreninger.

Systemet skal hjælpe med:

- Møder
- Dagsordener
- Referater
- Historik
- Opgaver
- Årshjul

AI skal:

- Foreslå dagsordenpunkter
- Oprette opgaver automatisk
- Huske tidligere beslutninger
- Hjælpe med opfølgning

## MVP

- Organisationer
- Udvalg
- Møder
- Dagsordenpunkter
- Referater
- AI-opgavegenerering
- Opgaveboard

## Tech Stack

- Next.js
- TypeScript
- Tailwind
- Supabase
- PostgreSQL
- OpenAI
