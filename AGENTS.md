# AGENTS.md

## Project Rules

Use:
- Next.js
- TypeScript
- Tailwind
- Supabase
- PostgreSQL

## Architecture

Agenda Item is the central entity.

Relationships:

Meeting -> Agenda Item -> Notes -> Tasks

## AI First

AI must:

- Create tasks from meeting notes
- Suggest agenda items
- Suggest historical context
- Help create job cards

## Development Principles

- Build flows, not features
- Mobile friendly
- Reusable components
- Strong typing
